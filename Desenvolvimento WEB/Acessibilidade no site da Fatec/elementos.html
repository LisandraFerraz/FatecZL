    Definição

    O doctype específica qual a versão de (X)HTML que a página está usando. Os elementos que formam o conteúdo da página são postos entre a tag de abertura "html" e a tag de fechamento "html". O elemento "html" é também conhecido como o elemento raiz. "html" O elemento HTML "html" (ou HTML root element) representa a raiz de um HTML ou XHTML documento. Todos os outros elementos devem ser descendentes desse elemento. Confira a tabela abaixo para mais detalhes:

    Elemento O que é
    Header É utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos de h1 a h6, até elementos para representar imagens, parágrafos ou mesmo listas de navegação.
    Nav Nav representa uma seção da página que contém link para outras páginas ou partes do mesmo website.
    Section Define um bloco ou um grupo de um assunto específico. É importante entender que a section agrupa diversos elementos que tenham relação entre si.
    Article É usado para marcar os conteúdos das páginas “estáticas”, tais como, as páginas About e Contact uma vez que o elemento
    popde ser usado para marcar “qualquer outro item independente de conteúdo”.
    Aside Define um bloco de conteúdo que faz referência ao conteúdo principal que o cerca, pode estar ou não em uma barra lateral (sidebar) exibindo informações contextuais.
    Footer Marca a área inferior, normalmente conhecida como rodapé, do conteúdo geral do documento ou do conteúdo de uma seção específica a qual ele está subordinado.
